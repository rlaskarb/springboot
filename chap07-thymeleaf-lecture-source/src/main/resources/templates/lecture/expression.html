th:block

HTML 출력 th:text, th:utext, th:value 

        서버에서 넘기는 값을 html 태그에 표현할 때 th:text, th:utext 를 사용할 수 있다.
        th:text 는 html 태그를 단순 문자열로 처리하지만, th:utext 는 html 태그 자체로 인식을 해준다.
        th:value 는 태그의 value 값을 지정할 수 있다.

표현식 인라인 표현식 [[...]],[(...)]

변수 표현식 값을 html 에 직접 사용하기 위해서 th:text 와 동일한 기능인 [[...]] 을 사용할수 있고, th:utext와 동일한 기능인 [(...)] 을 사용할수 있다.

  • [[${hello}]]
  • [$(hello)]
  • [[${hello}]]
  • [(${hello})]
